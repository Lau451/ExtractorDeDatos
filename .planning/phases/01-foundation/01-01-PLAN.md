**2. src/core/job_store.py** — In-memory job store with asyncio.Lock:
- `Job` dataclass with fields: `job_id: str`, `status: Literal["pending", "processing", "complete", "error"]`, `raw_text: Optional[str]`, `error_code: Optional[str]`, `error_message: Optional[str]`, `created_at: datetime`, `updated_at: datetime`
- `JobStore` class with `_store: dict[str, Job]` and `_lock: asyncio.Lock`
- Methods: `async create(job_id) -> Job`, `async get(job_id) -> Optional[Job]`, `async set_status(job_id, status)`, `async set_complete(job_id, raw_text)`, `async set_error(job_id, error_code, error_message)`
- All methods acquire `self._lock` before mutating `_store`
- Module-level singleton: `job_store = JobStore()`
- Use `datetime.utcnow()` for `created_at` and `updated_at` (update `updated_at` on every state change)
- Import `JobStatus = Literal["pending", "processing", "complete", "error"]` as a type alias

**3. src/api/models.py** — Pydantic response models:
```python
from pydantic import BaseModel
from typing import Optional

class ExtractResponse(BaseModel):
    job_id: str
    status: str  # always "pending" on POST /extract

class JobResultPayload(BaseModel):
    raw_text: str

class JobResponse(BaseModel):
    job_id: str
    status: str
    result: Optional[JobResultPayload] = None
    error_code: Optional[str] = None
    error_message: Optional[str] = None

class ErrorResponse(BaseModel):
    error: str
    message: str

class HealthResponse(BaseModel):
    status: str
```

**4. src/api/routes/health.py** — GET /health endpoint:
```python
from fastapi import APIRouter
from src.api.models import HealthResponse

router = APIRouter()

@router.get("/health", response_model=HealthResponse)
async def health():
    return {"status": "healthy"}
```

**5. src/main.py** — FastAPI app factory and uvicorn entry point:
```python
from fastapi import FastAPI
from src.api.routes.health import router as health_router

app = FastAPI(title="DocExtract", version="0.1.0")
app.include_router(health_router)

if __name__ == "__main__":
    import uvicorn
    from src.core.config import settings
    uvicorn.run("src.main:app", host=settings.api_host, port=settings.api_port, reload=True)
```
