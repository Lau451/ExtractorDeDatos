    **Step 4: Create `src/export/formatters.py`** with:

    1. Shared helper `_make_csv_bytes(headers: list[str], rows: list[list]) -> bytes`:
       - Create `io.StringIO` buffer
       - Write BOM character `"\ufeff"` to buffer
       - Create `csv.writer(buf)`, write header row, write all data rows
       - Return `buf.getvalue().encode("utf-8")`

    2. Helper `_format_line_item_type(model_class, item_model_class, extraction_result: dict) -> bytes`:
       - `result = model_class.model_validate(extraction_result)`
       - `header_fields = [f for f in model_class.model_fields if f != "line_items"]`
       - `item_fields = list(item_model_class.model_fields.keys())`
       - `columns = header_fields + item_fields`
       - `header_values = [getattr(result, f) for f in header_fields]`
       - If `not result.line_items`: single row = `header_values + [None] * len(item_fields)`
       - Else: one row per item = `header_values + [getattr(item, f) for f in item_fields]`
       - Replace None with `""` in every cell: `[("" if v is None else v) for v in row]`
       - Return `_make_csv_bytes(columns, rows)`

    3. Helper `_format_header_only_type(model_class, extraction_result: dict) -> bytes`:
       - `result = model_class.model_validate(extraction_result)`
       - `fields = list(model_class.model_fields.keys())`
       - `row = [("" if getattr(result, f) is None else getattr(result, f)) for f in fields]`
       - Return `_make_csv_bytes(fields, [row])`

    4. Five public functions:
       - `format_purchase_order(extraction_result: dict) -> bytes`: calls `_format_line_item_type(PurchaseOrderResult, POLineItem, extraction_result)`
       - `format_invoice(extraction_result: dict) -> bytes`: calls `_format_line_item_type(InvoiceResult, InvoiceLineItem, extraction_result)`
       - `format_supplier_comparison(extraction_result: dict) -> bytes`: calls `_format_line_item_type(SupplierComparisonResult, SupplierRow, extraction_result)`
       - `format_tender_rfq(extraction_result: dict) -> bytes`: calls `_format_header_only_type(TenderRFQResult, extraction_result)`
       - `format_quotation(extraction_result: dict) -> bytes`: calls `_format_header_only_type(QuotationResult, extraction_result)`

    5. `FORMATTER_REGISTRY: dict[str, Callable[[dict], bytes]]` mapping all 5 doc_type strings to their formatter functions. Keys MUST match SCHEMA_REGISTRY keys exactly: `"purchase_order"`, `"tender_rfq"`, `"quotation"`, `"invoice"`, `"supplier_comparison"`.
