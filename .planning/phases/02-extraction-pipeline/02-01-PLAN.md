      NOTE: `extraction_result` type is `Optional[dict]` (plain dict from Pydantic `.model_dump()`), not a Pydantic model. Import `dict` is from builtins, not typing.

   c. Add two new methods to `JobStore` (follow the same `async with self._lock` + `datetime.utcnow()` pattern as `set_complete`/`set_error`):
      ```python
      async def set_doc_type(self, job_id: str, doc_type: str) -> None:
          async with self._lock:
              job = self._store.get(job_id)
              if job:
                  job.doc_type = doc_type
                  job.updated_at = datetime.utcnow()

      async def set_extraction_result(self, job_id: str, result: dict) -> None:
          async with self._lock:
              job = self._store.get(job_id)
              if job:
                  job.extraction_result = result
                  job.status = "complete"
                  job.updated_at = datetime.utcnow()
      ```

   d. Keep ALL existing methods (`create`, `get`, `set_status`, `set_complete`, `set_error`) unchanged.
